2. Implicits and helper methods

Now that we know how to use the typeclass pattern. What else can we do to improve our code?

a. Implicits

We can use implicits to provide the instances of the typeclass. Instead of providing the parameters enc in the WriteStuff object, we can implicitly pass the corresponding ByteEncoder class with the implicit keyword, 

trait ByteEncoder[A] {
  def encode(a: A): Array[Byte]
}

trait WriteStuff {
  def write[A](obj: A)(implicit enc: ByteEncoder[A]): Unit
}

or defining A as of type ByteEncoder, 

trait ByteEncoder[A] {
  def encode(a: A): Array[Byte]
}

trait WriteStuff {
  def write[A: ByteEncoder](obj: A) : Unit
}
